# 更新履歴 (Changelog)

すべての変更点は [Keep a Changelog](https://keepachangelog.com/ja/1.1.0/) に準拠して記録されます。

## [0.2.0.0] - 2026-04-09

### ✨ 新機能 (Added)
- **認証の永続化**: Better Auth を導入し、`userToken` を使用した永続的なログインに対応しました。
- **動的アイコン更新**: Hubとの接続状態に応じて Stream Deck のボタンアイコンがリアルタイムに更新されるようになりました。
- **Geminiモデル選択**: 設定画面から使用するモデルを選択できるようになりました。

### ⚙️ 変更 (Changed)
- **リブランディング**: サービス名を "Mutsuna Voice Deck" へ統一し、Alexaへの直接的な依存表記を整理しました。
- **設定UIの刷新**: プロパティインスペクタ（設定画面）のレイアウトを改善し、使い勝手を向上させました。

### 🐞 修正 (Fixed)
- 設定画面でのAPIキー伏字表示の切り替え不具合を修正しました。
- マニフェスト内での名称衝突によるエラーを解消しました。

## [0.1.0.0] - 2026-04-08

### 🎉 初期リリース (Initial Release)
- **基盤構築**: Cloudflare Workers を利用したリレーサーバーのインフラを構築しました。
- **ボイスブリッジ**: Amazon Alexa から Stream Deck への基本的なコマンド送受信機能を実装しました。
- **Hub連携**: サーバー側との接続状態の監視およびステータス表示のUIを実装しました。
- **ライセンス**: 公開用リポジトリに MIT ライセンスを適用して配布を開始しました。
